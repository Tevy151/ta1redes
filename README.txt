Para ejecutar el programa, se necesitan dos terminales: una para el servidor y otra para el cliente. 
Primero, se debe ejecutar la terminal del servidor con go "run server.go", y luego, en la terminal del cliente, se ejecuta "go run cliente.go 127.0.0.1:1234".
Una vez que el cliente se conecte al servidor, se le proporcionarán algunas preguntas que deberá responder. Al finalizar, se le mostrará un puntaje final.
